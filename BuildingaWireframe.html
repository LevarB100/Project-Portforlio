Project Description

The purpose of this project is the familiarize the student with initial HTML and CSS methods of webpage design. The student uses HTML tags such as paragragh tags, header tags and CSS styling commands such as "float-left" , "margin" and "padding" to style the page as required.


Motivation

The purpose of the project is to familarize the the student with HTML and CSS styling procedures. The student will gain a basic understanding of how the dynamics of HTML and CSS design can be performed.


Results

At the end of this assignment the student will have designed a single web page that consist of multiple blocks laid out ona single background. The student will have a configuration of blocks centered on the webpage and arranged in a specific pattern that required the use of specific HTML and CSS methods to achieve.


Challenges of the Project

The main challenge of this project was to properly align the blocks centered in the middle of the page. Understanding the dynamics of things such as the margin, padding and border of a div item on a page is also a challenge when designing.


Improvements

An improvement I would like to make next time is the time I took to complete this assignment. I would also like to vary the color of the page as well.
